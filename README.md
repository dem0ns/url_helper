# URL Helper

在线 URL 参数编辑工具，纯前端实现，无需后端。

## 功能

### URL 解析
- 自动解析 URL 参数
- 支持自定义 scheme（如 `myapp://`）
- 实时解析，输入即生效

### 参数编辑
- 编辑参数值（支持 JSON 自动格式化）
- 添加新参数
- 删除参数
- 参数拖拽排序

### 批量操作
- 批量替换参数值
- 一键复制 URL
- 一键清空
- 一键在新标签页打开

### 二维码
- 自动生成当前 URL 二维码
- 点击放大查看
- 跟随 URL 实时更新

### 夜间模式
- 支持深色主题
- 自动保存偏好

## 使用

直接打开 `index.html` 即可使用。
